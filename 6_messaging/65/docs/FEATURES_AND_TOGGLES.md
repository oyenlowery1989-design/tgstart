# Features and Toggles

| Toggle | Description |
|--------|------------|
| toggle_mirror_new | Mirror new messages |
| toggle_edits | Track edits |
| toggle_deletes | Track deletions |
| toggle_admin | Track admin changes |
| toggle_restrict | Track bans/restrictions |
| toggle_invites | Track invite creation & joins |
| toggle_bots | Detect bot adds/removals |
| toggle_bio_worker | Background bio fetching |
| diff_mode | word / char |
